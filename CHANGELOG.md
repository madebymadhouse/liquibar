# Changelog

## [0.1.0] - 2026-05-20

### Added
- Initial release
- Liquidock runtime core (`packages/core`) - runtime contracts and state orchestration
- Liquid glass surface layer (`packages/glass`) - early visual layer
- React bindings (`packages/react`) - React bindings for the dock
- Tools package (`packages/tools`) - first-party tools and tool helpers
- Playground app (`apps/playground`) - minimal browser-viewable prototype demonstrating tool activation, attached surfaces, single-overlay model, draggable dock positioning, and hover expansion
